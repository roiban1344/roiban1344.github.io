### Matching with `Option<T>`

```rust
    fn plus_one(x: Option<i32>) -> Option<i32> {
        match x {
            None => None,
            Some(i) => Some(i + 1),
        }
    }

```

Variantが「実体」なのかそうでないのかでやや捉え難さを感じる．
